# @template/web

Minimal Vite + React + TypeScript app. `src/app.tsx` demonstrates the template's
conventions: kebab-case filenames, a colocated CSS Module (`app.module.css`) instead of
inline styles, and named (not default) component exports.

- `pnpm --filter @template/web dev` — start the Vite dev server
- `pnpm --filter @template/web build` — typecheck (`tsc -b`) then build with Vite
